# Task 2 - Intermediate Python Projects

This folder contains two intermediate-level Python projects completed as part of the **ShadowFox Python Programming Internship**.

---

## Project 1: Hangman Game

### Description

A console-based Hangman game where the player guesses a randomly selected word one letter at a time. The player has a limited number of lives to guess the word correctly.

### Features

* Random word selection
* Input validation
* Tracks guessed letters
* Displays remaining lives
* Win/Lose conditions

### Technologies Used

* Python
* Random Module

### Concepts Used

* Loops
* Conditional Statements
* Lists
* Strings
* Functions
* User Input

### Run the Project

```bash
python hangman.py
```

## Project 2: Web Scraper

### Description

A Python web scraper that extracts quotes and their authors from **https://quotes.toscrape.com** using the Requests and BeautifulSoup libraries.

The scraped data is displayed in the terminal and saved to a `quotes.txt` file.

### Features

* Sends HTTP requests
* Parses HTML pages
* Extracts quotes and authors
* Saves scraped data into a text file

### Technologies Used

* Python
* Requests
* BeautifulSoup4

### Concepts Used

* Web Scraping
* HTML Parsing
* File Handling
* Loops
* HTTP Requests

### Run the Project

```bash
python web_scraper.py
```
